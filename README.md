# VueJS with StyleX

Experiment project integrate Vue js with StyleX Facebook.
Forked from : [reacttips-dev/stylex](https://github.com/reacttips-dev/stylex)

Dependencies :

-   NodeJS <18.0.0
-   vue 2.6.11
-   stylex 0.1.0-beta.0
-   stylex-webpack-plugin 0.1.0-beta.0

## Development

```bash
# yarn

yarn install
yarn build
yarn serve
```
